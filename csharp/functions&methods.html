C# Functions & Methods — Easy Guide

Understand functions and methods in simple words: they are blocks of code you can call to do work, reuse logic, and keep programs clean.

1) What is a Function?

A function is a block of code that can run when called. It can take inputs (parameters) and optionally return a value.

using System;

class Program
{
    // Function: Adds two numbers and returns the result
    static int Add(int a, int b)
    {
        return a + b;
    }

    static void Main()
    {
        int sum = Add(5, 7);
        Console.WriteLine($"Sum = {sum}");
    }
}

2) Method vs Function

  • Function: general term for reusable code block.
  • Method: a function that belongs to a class or object (in C#, almost everything is inside a class, so we usually say "method").
class Calculator
{
    // This is a method because it belongs to the Calculator class
    public int Multiply(int x, int y)
    {
        return x * y;
    }
}

class Program
{
    static void Main()
    {
        var calc = new Calculator();
        Console.WriteLine(calc.Multiply(3, 4));
    }
}

3) Method Types

Static Method

Belongs to the class itself, not an object. Call with ClassName.Method().

class MathHelper
{
    public static double Square(double n) => n * n;
}

Console.WriteLine(MathHelper.Square(5));

Instance Method

Belongs to an object. Call with object.Method().

class Person
{
    string name;
    public Person(string name) { this.name = name; }
    public void SayHello() => Console.WriteLine($"Hi, I am {name}");
}

var p = new Person("Alex");
p.SayHello();

4) Parameters

Methods can take different kinds of parameters:

  • By Value (default) — a copy is passed.
  • ref — pass by reference, changes affect the caller.
  • out — must be assigned inside method.
  • Optional — can have default values.
  • Params — pass variable number of arguments.
class Demo
{
    public void ShowParams(params int[] numbers)
    {
        Console.WriteLine("Count: " + numbers.Length);
    }
}

class Program
{
    static void Main()
    {
        var d = new Demo();
        d.ShowParams(1, 2, 3, 4); // variable number of inputs
    }
}

5) Return Types

A method can:

  • Return void → nothing.
  • Return a value → int, string, etc.
  • Return a Task or Task<T> for async methods.
static void SayHello() => Console.WriteLine("Hello");

static string GetName() => "Alex";

static async Task DelayPrint()
{
    await Task.Delay(1000);
    Console.WriteLine("Done waiting");
}

6) Method Overloading

Same method name, but different parameter lists.

class Printer
{
    public void Print(string text) => Console.WriteLine(text);
    public void Print(int number) => Console.WriteLine(number);
}

var p = new Printer();
p.Print("Hello");
p.Print(42);

7) Local Functions

You can define a function inside another function (helps keep code organized).

static void Main()
{
    int Square(int x) => x * x; // local function

    Console.WriteLine(Square(4));
}
